# DSA_JAVA

A Java practice repository focused on Data Structures and Algorithms problems.

## Repository Structure

This repository is organized topic-wise. Each folder contains standalone Java programs for practice, such as:

- `Array`, `String`, `Sorting`, `Recursion`, `BackTracking`
- `LinkedList`, `Stack`, `Queue`, `Map`, `Hashing`
- `BST`, `graph`, `TwoPointer`, `Sliding Window`
- `LeetCode`, `Basic_Programs`, `OOP`, `Java_GUI`, and more

## How to Run

Programs are mostly independent Java files. Compile and run any file directly:

```bash
javac <FileName>.java
java <ClassName>
```

Example:

```bash
cd Array
javac Array1.java
java Array1
```
